# Proyecto Final - Análisis del Titanic

Este proyecto presenta un análisis del dataset Titanic utilizando el lenguaje R. 
Se aplican técnicas de limpieza de datos, contraste de hipótesis, regresión lineal 
y regresión logística para analizar los factores que influyeron en la supervivencia 
de los pasajeros.

## Contenido del proyecto

- Código de análisis en R
- Visualizaciones de datos
- Informe del análisis
- Dashboard interactivo

## Objetivo

Explorar los factores que influyeron en la supervivencia de los pasajeros del Titanic 
mediante técnicas de análisis estadístico y visualización de datos.

## Tecnologías utilizadas

- R
- ggplot2
- Shiny
